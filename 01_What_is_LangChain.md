# What is LangChain?
LangChain is an open source framework that helps in building LLM based applications. It provides modular components, such as chatbots, question-answering systems, retrieval-arguments generation(RAG), autonomous, and more. 
1. Supports all the major LLMs.
2. Simplifies developing LLM based application.
3. Integration available for all major tools.
4. Open Source/Free/Actively developed.
5. Supports all major GenAI use cases.
